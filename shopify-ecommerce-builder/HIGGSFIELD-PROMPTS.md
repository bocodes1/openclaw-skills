## 🏷️ BRAND CONTEXT BLOCK
*(Paste this mental model into every prompt you write)*

- **Product:** Sleek white/silver AI-powered kitchen nutrition scale with a small embedded LCD display, app-connected via Bluetooth
- **Brand aesthetic:** Bright, clean, healthy, premium health-tech — think Apple meets a wellness/nutrition brand
- **Color palette:** Bright greens (#22c55e), crisp whites, soft light grays, warm natural light, occasional marble/wood textures
- **Target user:** Health-conscious adults 25–40, active lifestyle, clean eating, tech-savvy
- **Hero settings:** Modern minimalist kitchens, white marble countertops, natural blond wood surfaces, fresh whole vegetables and fruits as props, large bright windows

---

## ✅ UNIVERSAL IMAGE RULES
*Apply EVERY rule to EVERY prompt — no exceptions.*

1. **Text readability:** Any text visible in the scene (app screens, scale display, labels) must be **fully legible, crisp, high-contrast**. Spell out exact strings you want shown.
2. **Product is hero:** The NutriScale product must be **fully visible, sharply focused, centered or dominant** in the frame — never cropped, never blurred, never partially obscured.
3. **Lighting:** **Bright natural window light from the left** OR **clean studio lighting from upper-left** — soft shadows only, no harsh dark areas, no underexposed regions.
4. **Quality suffix:** End EVERY prompt with: `commercial product photography, 8K, hyperrealistic, shot on Phase One IQ4 150MP, ultra-sharp focus`
5. **Negative constraints:** Add to EVERY prompt: `NO motion blur, NO depth-of-field blur on product or text, NO obscured display, NO reflections covering text, NO dark shadows on scale`

## 🔑 PROMPT ENGINEERING RULES (META)

| Rule | Why It Matters |
|------|---------------|
| Always spell out EXACT display text | AI will invent illegible text otherwise |
| Name the camera/lens (Phase One IQ4) | Triggers hyperrealistic quality model behavior |
| Specify light direction | Controls shadow placement for clean product shots |
| State negative constraints explicitly | Prevents blur, dark images, cropped products |
| Keep product description consistent | "sleek white AI smart kitchen scale with embedded LCD display" — use this exact phrase every time |
| Specify perspective (flat lay / front angle / three-quarter) | Controls composition dramatically |
| Include context props | Vegetables, marble, wood make images feel real vs. sterile |

---

## 🚫 COMMON MISTAKES TO AVOID

- ❌ Vague size/composition → product ends up tiny in frame
- ❌ No lighting direction → dark muddy images
- ❌ Not specifying display text → illegible characters
- ❌ Forgetting negative constraints → motion blur ruins shots
- ❌ Over-cluttered prompts → model gets confused, loses product focus
- ❌ Not specifying phone model → generic/fake-looking screens
